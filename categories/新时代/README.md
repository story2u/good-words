# 分类 · 新时代（New Age）

## 定义

以**赛斯资料（Seth Material）**为核心圈的新时代/身心灵书系。赛斯资料过于深奥，单读原典容易迷路，故本分类把帮助理解赛斯的周边著作与互文的新时代经典一并纳入，按与赛斯的关联度分三个圈层（见 `books.md`）：

- **圈层 A · 赛斯原典**：赛斯口述、珍·罗伯兹（Jane Roberts）传述、罗伯·巴茨（Robert Butts）记录的书。
- **圈层 B · 赛斯体系周边**：珍·罗伯兹个人著作、亲历者回忆与练习手册、华人推广者（王季庆、许添盛等）的译介与解读。
- **圈层 C · 新时代经典（互文）**：与赛斯思想互相印证的其他传讯/灵性经典（与神对话、奇迹课程、欧林、一的法则等）。

## 收录标准

1. 圈层 A/B：与赛斯资料直接相关即收；
2. 圈层 C：须是公认的新时代代表作，且与赛斯核心观念（意识创造实相、当下威力之点、转世与超灵等）有明确互文；
3. 每本书入单时标注：原名、作者、译者/出版社（已知）、关联度（★1–3）、摘抄优先级（P1–P3）、状态。
