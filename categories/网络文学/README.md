# 分类 · 网络文学（Web Literature）

## 定义

以中文网络连载/出版的长篇类型小说为主，重点收**玄幻、武侠、仙侠**一脉里文字与思想兼优的作品。网文体量极大（动辄数百万字），单本通读不现实，故本分类的取向不是"复述故事"，而是从长卷中**淘洗可独立成立的好词好句**：金句格言、世界观设定、写景文笔。

首本：《剑来》（烽火戏诸侯）。

## 收录标准

1. 公认有代表性的中文网络/类型长篇（玄幻、武侠、仙侠、历史等），文字水准在同类中居上；
2. 能从中稳定提炼出"脱离剧情也成立"的好句——格言警句、设定妙笔、写景白描，三者至少居其一；
3. 每本书入单时标注：作者、版本（出版/连载来源）、体量（卷数/字数）、摘抄优先级（P1–P3）、状态；
4. 体量巨大者按 `HARNESS.md` G4 摘录克制（指导线 ≤40 卡或全书正文 2%，先到为准）从严把握——网文动辄数百万字，**40 卡几乎总是先到的硬约束**，故只做精选高光集，不做覆盖式摘抄。
