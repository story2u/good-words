# playbook/00 · 流程总览

## 状态机（每本书）

```
候选 ──01 入库──▶ 入库 ──02 摘抄──▶ 摘抄中 ──03 校对──▶ 校对中 ──通过──▶ 已完成
 ▲                                      │                  │
 └──────────── 暂停/退回 ◀──────────────┴──────────────────┘
```

状态唯一记录处：`categories/<分类>/books.md` 的状态列。

## 三步工作流

| 步骤 | 文档 | 产物 |
|---|---|---|
| 1 选书入库 | `01-book-intake.md` | `books/<书名>.md` 书目档案；books.md 状态→入库 |
| 2 阅读摘抄 | `02-excerpt.md` | `excerpts/<书名>.md` 候选摘抄卡（按批次追加） |
| 3 校对入库 | `03-review.md` | 卡片定稿 + 审校记录；books.md 状态→已完成 |

## 批次

长书分批：一个批次＝一个阅读范围（如「第 1–5 章」或「511–540 节」）。批次进度记在该书摘抄文件头部的批次表，当前批次同步到 `relay/HANDOFF.md`。

## 会话纪律

开工先读 `relay/HANDOFF.md` → 干活 → 收工更新 `relay/HANDOFF.md`。规则疑义回 `HARNESS.md`。
